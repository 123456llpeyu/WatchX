# WatchX
> 开源智能手表，高质量、流畅的动画效果，方便扩展功能的系统框架。

## UI框架
> [LittlevGL V6.1.2](https://github.com/littlevgl/lvgl)

## 功能
* 1.时间显示
* 2.温度显示
* 3.气压显示
* 4.海拔显示
* 5.秒表
* 6.亮度控制
* 7.时间设置
* 8.待续..
